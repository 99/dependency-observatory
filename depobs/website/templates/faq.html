How are packages scored?

Packages are scored based on
  • their npms.io scores
  • the number of packages they depend on
  • the number of active vulnerabilities in the package
  • the number of active vulnerabilities in the package's dependencies
  • the number of package maintainers
  • the number of package contributors
You can see the detailed breakdown of the point values for each attribute below:
